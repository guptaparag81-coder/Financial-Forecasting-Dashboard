# Financial Forecasting Dashboard

A Python-based forecasting project using ARIMA time-series modelling.

## Objective

The purpose of this project is to forecast future sales values using historical data.

The project demonstrates statistical forecasting techniques commonly used in business analytics, finance, economics, and demand prediction.

## Features

- Time-series forecasting
- ARIMA modelling
- Future value prediction
- Statistical analysis
- Data visualisation

## Technologies

- Python
- Pandas
- NumPy
- Statsmodels
- Matplotlib

## Applications

- Sales forecasting
- Demand prediction
- Revenue forecasting
- Business analytics
- Financial modelling
